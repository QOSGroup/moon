
# Validator
> validator基础信息和参与打块
* Uri:

   `/nodes/{chainId}/validators/{address}`

* Method:

    `GET`

* Param

| 参数名   |      类型      |是否必须|默认值|说明|
|----------|:-------------:|:-----:|:---:|:--:|
| chainId | string | yes | - | 链ID|
| address | string | yes | - | 验证者地址|
| minHeight | integer | no | - | 打块查询|
| maxHeight | integer | no | - | 打块查询|
