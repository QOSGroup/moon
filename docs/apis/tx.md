
# Tx
> 交易详情

* Uri:

   `/nodes/{chainId}/txs/{txId}`

* Method:

    `GET`

* Param

| 参数名   |      类型      |是否必须|默认值|说明|
|----------|:-------------:|:-----:|:---:|:--:|
| chainId | string | yes | - | 链ID|
| txId | integer | yes | - |qmoon中交易ID|
